Este proyecto corresponde al desarrollo de un servidor en PHP para la Web que 
permita una comunicación estable y segura entre el brazo ESE_GRS y los clientes 
de [ESE_GRS_App](https://github.com/Esteban1914/ESE_GRS-Class). Además soporta el
[ESE_GRSConnectionSimulator](https://github.com/Esteban1914/ESE_GRS_ConnectionSimulator)
